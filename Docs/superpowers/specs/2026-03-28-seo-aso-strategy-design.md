### Screenshot Design Rules

- First 3 screenshots are most critical (most users only see these)
- Large, readable caption text on every screenshot
- Background gradient: orange-red tones matching lava theme
- No device frames (modern trend: frameless)
- Portrait orientation (matches gameplay)

### Feature Graphic (Google Play, 1024x500)

- Center: ball character (most popular skin), rotating lava tower in background
- Title: "TowerMaze" large font + "Climb. Escape. Survive." tagline
- Lava effects and particles

### App Preview Video (iOS, 15-30s)

- Show core gameplay loop: swipe to climb → navigate maze → escape lava rush
- Open with the most dramatic moment (lava rush near-escape)
- End with skin unlock moment + "Download Free" CTA
- No voiceover, gameplay audio only + subtle music
- Portrait orientation, 1080x1920

## 4. Web Landing Page SEO

### Domain

- **Primary:** `towermaze.game` or `towermaze.io`
- **Brand:** `sezogames.com` as company site

### URL Structure

```
towermaze.game/          → Main page (EN)
towermaze.game/tr/       → Turkish version
```

### HTML Meta Tags

**EN:**
```html
<title>TowerMaze — Climb the Rotating Lava Tower | Free Mobile Game</title>
<meta name="description" content="Escape rising lava in TowerMaze! Navigate a rotating maze tower, unlock epic ball skins, and see how high you can climb. Free on iOS & Android.">
<meta name="keywords" content="tower maze game, lava climbing game, rotating maze, ball climbing game, free mobile game, endless tower, maze puzzle game">
```

**TR:**
```html
<title>TowerMaze — Dönen Lav Kulesine Tırman | Ücretsiz Mobil Oyun</title>
<meta name="description" content="TowerMaze'de yükselen lavdan kaç! Dönen labirent kulesinde tırman, epik top skinleri aç ve ne kadar yükseğe çıkabileceğini gör. iOS ve Android'de ücretsiz.">
<meta name="keywords" content="kule labirent oyunu, lav tırmanma oyunu, dönen labirent, top oyunu, ücretsiz mobil oyun, bulmaca oyunu">
```

### Landing Page Sections

| Section | Content | SEO Purpose |
|---------|---------|-------------|
| **Hero** | Game video/GIF + "Download Free" CTA (App Store + Google Play badges) | Reduce bounce, increase conversion |
| **Features** | 4 cards: Rotating Maze / Lava Rush / Epic Skins / Daily Missions — each H2 | Keyword-rich headings |
| **How to Play** | 3 steps: Swipe → Climb → Escape | "how to play tower maze" searches |
| **Gallery** | Screenshot carousel (same as store) | Image search traffic |
| **Reviews** | User reviews from stores (post-launch) | Social proof + fresh content |
| **Download** | App Store + Google Play buttons repeated | Bottom CTA |
| **Footer** | Privacy Policy, Terms, Contact, SezoGames link | Legal + store requirements |

### Technical SEO Checklist

- `hreflang` tags: `en` + `tr` cross-referenced
- Open Graph + Twitter Card meta tags for social sharing previews
- Page speed: static site (GitHub Pages or Netlify), < 2s load
- `robots.txt` + `sitemap.xml`
- Google Search Console + Bing Webmaster registration
- App Links: `/.well-known/assetlinks.json` (Android) + `apple-app-site-association` (iOS)

- "tower maze" / "maze tower"

## 8. Measurement & Iteration

### Tools

- **Google Play Console** — store listing experiments, acquisition reports, search terms
- **App Store Connect** — product page optimization, impression sources
- **Google Search Console** — landing page search performance
- **AppFollow or Sensor Tower** — keyword ranking tracking (free tier sufficient at launch)

### Post-Launch Optimization Schedule

| Timeframe | Action |
|-----------|--------|
| Week 1-2 | Collect baseline data: impressions, conversion rate, top search terms |
| Week 3 | First Google Play store listing experiment (icon or screenshot A/B test) |
| Week 4 | Revise keywords based on actual search term data from both stores |
| Month 2 | Screenshot A/B test round 2, description optimization based on conversion data |
| Monthly | Review keyword rankings, update long-tail keywords, refresh social content |

### Key Metrics to Track

- **Store conversion rate** (impressions → installs) — target: >30% for organic
- **Keyword ranking position** — track top 20 keywords weekly
- **Landing page CTR** — Google Search Console click-through rate
- **Social media referral installs** — UTM-tagged store links
